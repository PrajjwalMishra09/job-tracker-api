# Job Tracker

A web-based Job Application Tracker built using Java, Spring Boot, MySQL, Thymeleaf, and Hibernate.

## Features

* Add Job Applications
* View All Applications
* Update Application Status
* Delete Applications
* Store Data in MySQL Database
* Responsive UI using Bootstrap

## Technologies Used

* Java 24
* Spring Boot
* Spring Data JPA
* Hibernate
* MySQL
* Thymeleaf
* Bootstrap 5
* Maven
* Git & GitHub

## Application Status Workflow

* Applied
* Interview
* Offer
* Rejected

## Run Locally

1. Clone the repository
2. Configure MySQL
3. Update `application.properties`
4. Run:

```bash
mvn spring-boot:run
```

5. Open:

```text
http://localhost:8080
```
